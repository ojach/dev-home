使い方ガイド

Step1
OJapp でアプリ素材を作る

1. OJapp にアクセスします。
2. アイコン画像を選択します。
3. アプリ名とリンク（URL）を入力します。
4. Create App 🎉 をタップします。
5. ページ下に生成される以下3つをコピーまたは保存します：

  • index.html
  • manifest.json
  • アイコン画像（PNG）

Step2
GitHub にアプリ用ページを作成

⚠️ GitHub アカウントが必要です。

1. GitHub にログインします。
2. 新しいリポジトリを作成（例：my-app-icon）。
3. Add file → Create new file を選択します。
4. ファイル名に index.html を入力し、生成したHTMLをペーストします。
5. 同様に manifest.json を作成して貼り付けます。
6. Add file → Upload files から PNG アイコンをアップロードします。

Step3
GitHub Pages で公開

1. リポジトリの Settings を開きます。
2. 左メニューの Pages を選択します。
3. Source を Deploy from branch または GitHub Actions に設定します。
4. 表示された URL を確認します（例：https://username.github.io/my-app-icon/）。

Step4
スマホで公開URLを開く

公開されたURLにアクセスすると：

  • 初回アクセス → 30秒カウントダウン画面
  • 自動でリンク先へ移動
  • ホーム画面追加にも対応（iPhone/Android）

Step5
30秒以内にブックマーク or ホーム追加

初回30秒以内に以下を行います：

  • ✔ ホームに追加
  • ✔ ブックマーク保存

これで、あなたのオリジナルショートカットが完成です！
以後はシンプル画面から2秒で自動ジャンプします。

お疲れさまです！🎉

OJapp と GitHub を組み合わせるだけで、完全オリジナルの “ホーム画面アプリ” を作成できます。

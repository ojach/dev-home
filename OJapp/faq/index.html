FAQ（よくある質問）

OJappを利用する上でよくある質問と回答をまとめました。

📱 ホーム追加・表示について

Q. 「ホーム画面に追加」が表示されません

A. iPhone では Safari のみ ホーム追加に対応しています。
Chrome や LINE ブラウザでは表示されません。

Q. ホームに追加したのにアイコンが反映されません

A. iOS のキャッシュが強いため、反映に時間がかかることがあります。
確実に反映したい場合は、ホームから一度削除して再度追加してください。

Q. 初回アクセスの 30 秒カウントダウンは何？

A. ホーム追加を案内するための仕組みです。
1度追加すると、次回以降はカウントダウンなしで開きます。

🖼 アイコン・画像について

Q. どんな画像をアップすればいいですか？

A. 推奨は 512×512px の PNG です。
それ以外でも自動調整されます。

Q. 写真フォルダから画像を選べますか？

A. はい、スマホから直接選択できます。

🌐 GitHub Pages について

Q. 公開URLはどこで確認できますか？

A. GitHub の Settings → Pages に表示されます。
一般的には以下の形式です：
https://{ユーザー名}.github.io/{リポジトリ名}/

Q. ページを消したい場合は？

A. GitHub 上で該当フォルダを削除すれば即反映されます。

❓ その他

Q. このURLは安全ですか？

A. GitHub Pages は公式のホスティングサービスで安全です。
OJapp はユーザーデータを収集しません。

Q. manifest.json を編集しても大丈夫ですか？

A. 表示名の変更などは問題ありませんが、
icons.src や start_url を変更すると動作しなくなる場合があります。
